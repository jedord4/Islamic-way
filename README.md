# Islamic Way App

Islamic Way is a beautifully designed, cross-platform mobile application built with **Flutter** that serves as a daily spiritual companion. It features an interactive learning module powered by **Google Gemini AI** to provide accessible, guided knowledge about Islam.

## Technical Overview

### 🚀 Built With
- **Flutter & Dart**: The core framework providing a seamless, cross-platform UI with smooth animations and robust state management.
- **Google Generative AI (Gemini)**: Integrates intelligent, contextual Q&A features to help users explore Islamic teachings efficiently. Responses are kept concise and beginner-friendly using customized system prompts.
- **Android Gradle**: Customized build configuration for reliable `.apk` packaging.

### ✨ Key Features
- **Dynamic Theming & Wallpapers**: Advanced UI dynamically adapts color schemes based on user-selected high-definition wallpapers.
- **Multi-language Support**: Hardcoded native support for languages including Arabic and Kinyarwanda, scaling state management robustly.
- **AI-Powered "Ask" Module**: Directly interfaces with the Gemini API to retrieve customized answers (kept under 100 words in easy-to-understand English).
- **Offline First Delivery**: Core content modules (Pillars of Islam, Daily Verses) are persistently available.

### 🛠 Deployment (GitHub Pages)
This repository contains the GitHub Pages deployment for the application landing page.
- **`index.html`**: The single-page landing site designed with a spiritual Emerald Green and Cream palette.
- **`Islamic_way_v1.zip`**: The distributable Android package compressed into a zip folder, ready for download.
- **`favicon.png`**: The application icon embedded seamlessly into the landing experience.

## Getting Started

To install the app on your Android device:
1. Download the `Islamic_way_v1.zip` from the GitHub Pages site and extract it.
2. Inside the extracted folder, you will find `Islamic_way_v1.apk`.
3. Allow installation from unknown sources in your Android security settings.
4. Install the extracted `.apk` and launch the application.
